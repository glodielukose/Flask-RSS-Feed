# Flask-RSS-Feed

**Description:**  
Une application web construite avec Flask pour lire et afficher des flux RSS.

## Fonctionnalités

- Lire des flux RSS depuis différentes sources.
- Affichage propre et organisé des articles.
- Interface utilisateur simple et intuitive.

## Prérequis

- Python 3.7 ou supérieur
- Flask

## Installation

1. Clonez le dépôt:
    ```bash
    git clone https://github.com/votre-utilisateur/Flask-RSS-Feed.git
    cd Flask-RSS-Feed
    ```

2. Créez un environnement virtuel et activez-le:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Sous Windows utilisez `venv\Scripts\activate`
    ```

3. Installez les dépendances:
    ```bash
    pip install -r requirements.txt
    ```

## Utilisation

1. Lancez l'application:
    ```bash
    flask run
    ```

2. Accédez à l'application via votre navigateur à `http://127.0.0.1:5000`.
